3. 공통 레이아웃

사용자 헤더

  • 좌측: 행정안전부 로고 + 시스템명 (클릭 시 지도 대시보드로 이동)
  • 우측 네비게이션: 지도 | 서비스 지원(hover 시 드롭다운: 공지사항, 자주 묻는 질문) | 관리 | 사용자 드롭다운
  • 관리 버튼: 관리자 권한 사용자에게만 표시됨. 일반 사용자(점검자, 열람자)에게는 숨김 처리
  • 사용자 드롭다운: 클릭 시 마이페이지, 로그아웃 메뉴 표시

관리자 헤더

  • 좌측: "관리자 포털" 타이틀 (클릭 시 관리자 대시보드로 이동)
  • 센터: 네비게이션 메뉴 (대시보드, 회원 관리, 접속 현황, 지역 이용 현황, 접속 로그, 게시판 관리 드롭다운)
  • 게시판 관리 드롭다운: hover 시 공지사항 관리, 자주 묻는 질문 관리 서브메뉴 표시
  • 우측: 지도 (사용자 대시보드로 이동) | 사용자 드롭다운 (마이페이지, 로그아웃)
  • 모바일(1024px 이하): 센터 네비게이션 숨김, 햄버거 버튼으로 사이드바 메뉴 표시

4. 인증

4.1 로그인

login.html

시스템 진입점. 아이디/비밀번호 인증을 통해 권한에 맞는 화면으로 진입한다.

관리자 점검자 열람자
login.html

주요 기능

  • 로그인 폼: 아이디(ID) 입력, 비밀번호 입력, 로그인 버튼
  • 에러 처리: 빈 필드 검증, 잘못된 자격증명 시 인라인 에러 메시지
  • 회원가입 링크: 로그인 폼 하단에 회원가입 페이지 이동 링크
  • 리다이렉트: 인증 성공 시 대시보드(dashboard.html)로 이동

UI 컴포넌트

  • 텍스트 입력 필드 (아이디, 비밀번호)
  • 로그인 버튼 (primary)
  • 회원가입 텍스트 링크
  • 에러 메시지 영역
반응형: 로그인 폼은 모든 해상도에서 중앙 정렬. 모바일에서 전체 너비 입력 필드.

4.2 회원가입

register.html

3단계 프로세스를 통한 회원가입. 약관 동의, 정보 입력, 가입 완료 순으로 진행한다.

비회원

Step 1 — 약관 동의

서비스 이용약관, 개인정보 처리방침 동의. 필수 약관 미동의 시 다음 단계 버튼이 비활성됩니다.

Step 2 — 기본정보 입력

이름, 이메일, 인증번호 확인. 필수 필드 미입력 시 에러 메시지가 표시됩니다. 이메일 인증번호 발송 후 확인 절차가 필요합니다.

Step 3 — 가입 완료

입력한 정보를 확인하고 가입 신청합니다. 확인 모달 → 성공 모달 순으로 표시되며, 로그인 페이지로 이동합니다.

주요 기능

  • Step 1 — 약관 동의: 서비스 이용약관, 개인정보 처리방침 동의 체크. 필수 약관 미동의 시 다음 단계 버튼 비활성
  • Step 2 — 기본정보 입력: 이름, 이메일, 인증번호 확인. 필수 필드 미입력 시 에러 메시지 표시
  • 이메일 인증: 인증번호 발송 및 확인 절차
  • Step 3 — 가입 완료: 확인/성공 모달, 로그인 페이지 이동

UI 컴포넌트

  • 스텝 인디케이터 (1/2/3 단계 표시)
  • 약관 동의 체크박스
  • 폼 입력 필드 (텍스트, 이메일, 비밀번호, 셀렉트)
  • 이메일 인증 버튼 + 타이머
  • 확인/성공 모달
반응형: 모바일에서 스텝 인디케이터는 축소형 표시. 입력 필드 1열 레이아웃.

5. 사용자 화면

5.1 지도 대시보드

dashboard.html

시스템의 핵심 화면. 전국 불법 시설물 현황을 지도 기반으로 시각화하며, 3단계 클러스터링으로 상세 조회가 가능하다.

관리자 점검자 열람자
dashboard.html

1단계 — 전국 파이차트 (기본 뷰)

줌 레벨 7 이하에서 시·도별 원형 차트로 걸침(오렌지)/구역내(빨강) 비율을 시각화합니다.

사이드바 — 시·도 클릭 상태

좌측 사이드바에서 시·도를 클릭하면 해당 지역이 확장되어 시·군·구 목록이 표시되고, 지도가 해당 영역으로 줌인됩니다.

2단계 — 클러스터링

시·도 선택 또는 줌인(줌 8+) 시 Leaflet MarkerCluster로 자동 클러스터링됩니다. 줌인할수록 클러스터가 분산되며, 걸침 포함은 오렌지, 구역내만은 빨간색으로 표시됩니다.

사이드바 — 시·군·구 클릭 상태

시·군·구를 클릭하면 해당 지역의 시설물 목록이 표시되고, 지도에 시설물 폴리곤이 렌더링됩니다.

사이드바 — 시설물 클릭 상태

시설물을 클릭하면 지도에서 해당 폴리곤이 하이라이트되고, 다크 글래스 팝업이 표시됩니다.

3단계 — 시설물 팝업

시설물 폴리곤 또는 목록 클릭 시 다크 글래스 스타일의 상세 팝업이 표시됩니다. 걸침(55%) 또는 구역내(100%) 유형, 하천등급, 하천명, 주소, 건물면적, 점유면적, 점유율 정보를 제공합니다.

주요 기능 요약

  • 3단계 클러스터링: 파이차트(시·도 레벨, 줌 7 이하) → MarkerCluster(시·군·구 레벨, 줌 8~14) → 폴리곤(개별 시설물, 줌 15+)
  • 좌측 사이드바: 지역별 현황 (걸침/구역내 분류), 정렬 기능(걸침순/구역내순/지역명순), 시설물 목록 스크롤
  • KPI 표시: 상단에 전국 현황 — 걸침 총합, 구역내 총합, 걸침+구역내 프로그레스 바
  • 시설물 팝업: 다크 글래스 스타일. 하천등급, 하천명, 주소, 건물면적, 점유면적, 점유율 표시. X 버튼으로 닫기
  • 시설물 폴리곤: 걸침(오렌지, 점선 테두리), 구역내(빨강, 실선 테두리). 클릭 시 하이라이트(흰색 테두리)
  • 베이스맵 전환: 위성(기본) / 일반 / 그레이 / 다크 배경지도 선택
  • 보안 서약서 모달: 데이터 다운로드 시 서약서 동의 + 요청명/활용기간/사용목적 입력
  • 줌아웃 시 자동 복귀: 줌 7 이하로 축소 시 파이차트로 자동 복귀, 클러스터 숨김
  • 지도 이동 시 폴리곤 갱신: 줌 15+ 에서 현재 화면 영역에 동적으로 폴리곤 생성. 팝업 열린 상태에서는 유지

검색 기능

우측 상단 검색 버튼 클릭 시 검색 오버레이가 열리며, 지역명/시설물명으로 검색 가능합니다.

검색어 입력 후 엔터 시 결과 목록이 표시됩니다. 결과를 클릭하면 해당 위치로 이동합니다.

배경지도 변경

우측 도구의 배경지도 버튼 클릭 시 위성/일반/그레이/다크 배경을 선택할 수 있습니다.

측정 도구

거리 측정, 면적 측정 등 지도 위에서 측정 기능을 제공합니다.

그리기 도구

지도 위에 점/선/면을 그릴 수 있는 도구입니다.

보안 서약서 모달 (내보내기 시)

데이터 다운로드 또는 지도 저장 시 보안 서약서 동의가 필요합니다. 서약 내용 확인, 동의 체크, 요청명/활용기간/사용목적 입력 후 실행합니다.

지도 저장 모달

보안 서약서 동의 후 지도 미리보기와 메모를 입력하여 지도를 PNG로 저장할 수 있습니다.

UI 컴포넌트

  • Leaflet 지도 (전체 화면 배경)
  • 파이차트 마커 (시·도 단위 집계)
  • MarkerCluster 그룹 (자동 클러스터링)
  • 폴리곤 오버레이 (시설물 경계, 걸침=오렌지 점선, 구역내=빨강 실선)
  • 좌측 사이드바 패널 (접기/펼치기, 정렬, 필터)
  • KPI 리본 바 (걸침 총합, 구역내 총합)
  • 우측 도구 모음 (검색, 배경지도, 측정, 그리기, 내보내기, 줌)
  • 글래스모피즘 다크 팝업 (시설물 상세)
  • 보안 서약서 모달 + 지도 저장 모달
반응형: 모바일에서 좌측 사이드바 → 바텀 시트로 전환. KPI는 리본 형태로 축소. 베이스맵 스위처는 축소 아이콘으로 표시.

5.2 공지사항

notice.html

공지사항 목록 및 상세 조회. 읽기 전용으로 관리 기능은 제공하지 않는다.

관리자 점검자 열람자
notice.html

상세 조회 화면

목록에서 공지사항을 클릭하면 읽기 전용 상세 화면으로 전환됩니다. 카테고리 배지, 제목, 등록일, 본문 내용이 표시되며, 첨부 파일이 있는 경우에만 파일 영역이 표시됩니다.

주요 기능

  • 목록 보기: 구분(긴급/일반/업무), 제목, 등록일 표시. 등록자는 숨김 처리
  • 상세 보기: 읽기 전용 모드. 카테고리 뱃지, 제목, 등록일, 본문 내용 표시
  • 첨부 파일: 파일이 등록된 경우에만 영역 표시. 파일명, 용량 표시. 미등록 시 숨김
  • 목록 복귀: "목록" 버튼으로 리스트 화면 이동

UI 컴포넌트

  • 공지사항 테이블 (구분, 제목, 등록일)
  • 카테고리 뱃지 (일반=파랑)
  • 상세 보기 카드
  • 첨부파일 다운로드 링크
  • 페이지네이션
반응형: 768px 이하에서 테이블 → 카드형 목록으로 전환.

5.3 자주 묻는 질문

faq.html

FAQ를 아코디언 형태로 제공. 카테고리별 분류 뱃지로 빠른 식별이 가능하다.

관리자 점검자 열람자
faq.html

아코디언 펼침 상태

질문을 클릭하면 답변이 펼쳐집니다. 카테고리 배지가 파란색으로 활성화되고, 화살표가 위로 회전합니다. 다른 항목을 클릭하면 기존 항목은 접힙니다.

주요 기능

  • 아코디언 목록: 질문 클릭 시 답변 영역 펼침/접힘. 한 번에 하나만 열림
  • 카테고리 뱃지: 시설물 조회, 지도/레이어, 계정/인증, 시스템 일반. 펼침 시 파란색 활성화
  • 등록일 표시: 각 항목에 날짜 표시, 작성자는 숨김
  • 화살표 애니메이션: 펼침 시 180도 회전

UI 컴포넌트

  • 아코디언 패널 (열기/닫기 토글)
  • 카테고리 뱃지 칩
  • 날짜 텍스트

관리자
admin_access.html

일간 차트 (기본값)

00시~23시까지 24개 막대로 시간대별 접속 수를 표시합니다. 기준일은 date picker로 선택합니다. 가장 많은 접속 시간대를 100%로 계산하여 막대 높이를 결정합니다.

주간 차트

선택한 주의 월요일~일요일까지 7개 막대로 요일별 접속 수를 표시합니다.

선택한 월의 1일~말일까지 막대를 표시합니다. 이번 달은 오늘까지만 데이터가 있으며, 기준일은 month picker로 선택합니다.

주요 기능

  • KPI 카드: 현재 접속자, 오늘 총 접속, 이번 주 접속, 이번 달 접속 (전주/전월 대비 증감 표시)
  • 필터: 조회 유형 버튼(일간/주간/월간), 기준일(일간·주간=date picker, 월간=month picker), 초기화/조회
  • 조회 버튼: 클릭 시에만 차트 갱신 (모드 전환만으로는 변경 안 됨)
  • 세로 막대 차트: 호버 시 반투명 툴팁(정확한 시각 + 접속 수)
  • 차트 헤더: 차트 타이틀 + 우측에 총 접속 건수 표시
  • 모바일(768px 이하): 일간 24시간 → 3시간 단위 8개 막대로 병합
반응형: 필터 토글 패널. 차트 가로 스크롤. KPI 카드 2열 → 1열. 모바일 일간 3시간 병합.

6.8 관리자 마이페이지

admin_mypage.html

사용자 마이페이지와 동일한 기능을 관리자 헤더/네비게이션 안에서 제공한다.

관리자
admin_mypage.html

주요 기능

  • 관리자 헤더: 관리자 포털 네비게이션 유지
  • 회원정보 카드: 사용자 마이페이지와 동일
  • 비밀번호 변경: 사용자 마이페이지와 동일 모달
  • 회원정보 수정: 사용자 마이페이지와 동일 프로세스
  • 최근 활동: 관리 활동 포함 이력 표시

UI 컴포넌트

  • 관리자 헤더 네비게이션
  • 정보 카드 (사용자 마이페이지와 동일)
  • 비밀번호 변경 모달
  • 회원정보 수정 폼
반응형: 사용자 마이페이지와 동일. 관리자 헤더는 햄버거 메뉴로 축소.
